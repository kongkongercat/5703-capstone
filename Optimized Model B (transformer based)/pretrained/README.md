# Pretrained Models

This folder is used to store pretrained model weights.

⚠️ Note:  
- Large model files (e.g., `.pth`, `.pt`, `.ckpt`) are **ignored by Git**.  
- Please download pretrained models separately from provided links or checkpoints.  
- Example:  
  - TransReID pretrained weights  
  - ViT/DeiT backbone checkpoints
