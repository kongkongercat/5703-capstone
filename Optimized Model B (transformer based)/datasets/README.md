# Datasets

This folder is used to store datasets required for training and evaluation.

⚠️ Note:  
- Large dataset files are **ignored by Git** (see `.gitignore`).  
- Please download the datasets separately (e.g., from Google Drive, Kaggle, or Hugging Face).  
- Typical content may include:  
  - VeRi-776  
  - VehicleID  
  - Custom processed data
